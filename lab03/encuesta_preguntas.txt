SECCIÓN 4 — DISPOSICIÓN A PAGAR
(La más importante para el análisis de mercado)
----------------------------------------------------------------

P12. ¿Qué modelo de pago preferiría para una plataforma así?
   ( ) Mensual por docente (pago individual)
   ( ) Licencia anual por institución (pago institucional)
   ( ) Pago único de por vida
   ( ) No pagaría por ningún modelo

P13. ¿Cuánto estaría dispuesto a pagar MENSUALMENTE por
     docente para acceder a la plataforma completa?
   ( ) Menos de S/. 20 al mes
   ( ) Entre S/. 20 y S/. 35 al mes
   ( ) Entre S/. 36 y S/. 55 al mes
   ( ) Entre S/. 56 y S/. 80 al mes
   ( ) No pagaría por ningún monto

     NOTA: Esta pregunta define el precio ponderado del estudio.
     El rango S/. 15–35/mes es el más frecuente entre docentes
     peruanos según estudios comparables (Kahoot! Pro: ~S/. 37,
     ClassDojo Plus: ~S/. 25 al mes por docente).

P14. Si su institución pagara la suscripción,
     ¿cuánto considera razonable que pague anualmente?

NOTA METODOLÓGICA:
Las preguntas P4, P7 y P13 son los filtros principales para
el modelo de segmentación en cascada:
  - P4  →  Mercado Disponible  (trabaja actualmente)
  - P7  →  Mercado Efectivo    (percibe baja motivación)
  - P13 →  Mercado Objetivo    (dispuesto a pagar)

El precio ponderado se calcula sobre los respondentes de P13
que eligieron un rango de pago (excluye "No pagaría").
